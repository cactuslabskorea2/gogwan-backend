# MBTI 이미지 폴더

이 폴더에 미드저니에서 생성한 MBTI별 이미지를 넣어주세요.

## 파일명 형식

각 MBTI 유형에 맞게 다음 형식으로 이미지를 저장해주세요:

```
INTJ.png  (또는 .jpg)
INTP.png
ENTJ.png
ENTP.png
INFJ.png
INFP.png
ENFJ.png
ENFP.png
ISTJ.png
ISFJ.png
ESTJ.png
ESFJ.png
ISTP.png
ISFP.png
ESTP.png
ESFP.png
```

## 필요한 이미지 (총 16개)

1. **INTJ** - 전략가
2. **INTP** - 논리술사
3. **ENTJ** - 통솔자
4. **ENTP** - 변론가
5. **INFJ** - 옹호자
6. **INFP** - 중재자
7. **ENFJ** - 선도자
8. **ENFP** - 활동가
9. **ISTJ** - 현실주의자
10. **ISFJ** - 수호자
11. **ESTJ** - 경영자
12. **ESFJ** - 집정관
13. **ISTP** - 장인
14. **ISFP** - 모험가
15. **ESTP** - 사업가
16. **ESFP** - 연예인

## 권장 사양

- 형식: PNG 또는 JPG
- 크기: 800x800 픽셀 이상
- 용량: 500KB 이하 (웹 최적화)
